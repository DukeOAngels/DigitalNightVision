[ 🛡️ SECURITY & TRUST ]
--------------------------------------------------------------------
Is this a virus? 

NO. 
Because this is an unsigned .exe, Windows may flag it. It is safe!
The app triggers alerts because it requires these features to work:
1. DXCAM: High-speed screen capture to apply the NVG filter.
2. PYNPUT: Listens for Mouse/Keyboard buttons to toggle the app.
3. WDA_EXCLUDE: Prevents the app from "recording itself" (Stealth).

[ 🛠️ TROUBLESHOOTING ]
--------------------------------------------------------------------
* SCREEN IS BLACK?
  Ensure game is in "Borderless Windowed" or "Windowed" mode. 
  "Exclusive Fullscreen" might block the overlay.

* USING FULLSCREEN? 
  If you must use Fullscreen mode, ensure "Disable Fullscreen Optimizations" 
  is UNCHECKED in your game's .exe compatibility properties.

* APP CRASHES ON START?
  - Ensure "CUDA_Bin" folder is in the same directory as the .exe.
  - Update NVIDIA drivers via GeForce Experience.

* PERFORMANCE LAGGING?
  - This app uses the GPU. If your FPS drops, lower your in-game 
    graphics settings to free up resources for the NVG math.
  - Laptop users: Must be plugged into a wall outlet for full power.
